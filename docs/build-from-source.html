Building from source

Requirements

  • Java 8 JDK

  • Maven 3

Checkout the source code

Fork me on GitHub: https://github.com/bloomreach-forge/hst-spring-security.

Build the project using Maven: 

$ mvn install

Build all the reports and the site (requires Maven 3): 

$ mvn site
